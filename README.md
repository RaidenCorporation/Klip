# How To Use RockZ™
#### On Android:
**Option 1:** Installing the ZIP (not recommended for regular users)

**Option 2:** Installing the single APK

## Option 1
1. Look for the full ZIP from the latest release on the Releases tab.
2. Scroll down on the latest release to see “Assets”.
3. In the Assets section, look for RockZ.v[version number].zip and click it.
4. It will install.
5. Go to your regular Files app.
6. Press the 3 vertical dots ⋮ on the file, or, hold on the ZIP and press Extract.
7. It will be extracted. Look in the folder for app-release.apk or any file ending in ".apk".
8. Once you found the APK file, press the 3 dots ⋮ or hold the file and press "Install".
9. It will be installed to your device. Enjoy the fun!
