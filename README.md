# How To Use RockZ™
#### On Android:

## Option 1
1. Look for the full ZIP from the latest release on the Releases tab.
2. Scroll down on the latest release to see “Assets”.
3. In the Assets section, look for app-release.apk and click it.
4. It will install.
5. Go to your regular Files app.
6. Look for app-release.apk.
7. Once you found the APK file, press the 3 dots ⋮ or hold the file and press "Install".
8. It will be installed to your device. Enjoy the fun and coin collecting!
