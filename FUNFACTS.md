# Fun Fact!
Fun Fact, everyone! First, Klip™ was *originally* named **RockZ™**. I had to change it because other brands and companies were using it and I didn't want to seem like a copycat even though I'm not.

So, I want to tell you a fun fact about the new name.

Did you know that I renamed it to **Klip™** because actually, **klip (pronounced kluh-pp)** is a real, Akrikaans word translating to *stone* (or rock). I choosed this because I thought it would be cool to use another language's word to use it as a brand. 

# Pronunciation
**Klip™** is pronounced as **kluh-pp**. Don't confuse it with the English word "clip" — that's a whole unaffiliated different word.
